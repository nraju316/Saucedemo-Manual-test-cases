# SauceDemo Manual Test Cases

Comprehensive manual test case suite covering the complete SauceDemo application workflow, including Login, Inventory, Cart, Checkout, and End-to-End scenarios.

The Excel workbook is organized into multiple dedicated sheets/modules for better traceability, structured coverage, and maintainability of positive, negative, validation, and exploratory test scenarios.

## Modules Covered

- Login Page
- Inventory Page
- Cart Page
- Checkout Page
- End-to-End Test Scenarios

## Test Coverage Includes

- Positive test scenarios
- Negative test scenarios
- Validation test cases
- Functional workflows
- End-to-End user journeys
- Boundary and edge case validations
- Exploratory testing scenarios
